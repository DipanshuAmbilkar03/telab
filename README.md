## Repository Structure (Short Version)

- `AI/` -> Artificial Intelligence practicals
- `ANN/` -> ANN practical notebooks
- `CN/` -> Computer Networks practicals and packet captures
- `Data Science/` -> Data Science labs + datasets
- `DBMS/` -> SQL and DBMS lab work

In short: chaos hai, but categorized chaos hai.
